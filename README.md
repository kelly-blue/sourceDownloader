Description:
A Chrome extension for downloading all website source files
visible under the DevTools Sources tab, preserving folder structure.
Now includes a live progress bar and modern toast notification.

Usage:
1. Open Chrome → chrome://extensions/
2. Enable Developer Mode.
3. Click "Load unpacked" and select this folder.
4. Open any website.
5. Click the extension icon → "Start Download".
6. Choose where to save files.

You'll see a progress bar and a modern toast message when done.
